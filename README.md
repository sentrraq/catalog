**Overview**
Sentraq is a professional digital solutions provider website showcasing IT services, products, and contact information with a modern dark-themed design.

**Features**
Modern Dark UI/UX - Elegant dark theme with neon accents
Responsive Design - Fully responsive across all devices
Interactive Elements - Custom cursor, animations, and hover effects
Product Showcase - Interactive product sliders with details
Contact Section - Multiple contact methods with hover effects
Performance Optimized - Fast loading and smooth animations
Accessibility - Screen reader friendly with proper semantics

**Tech Stack**
Frontend: HTML5, CSS3, JavaScript (ES6+)
Styling: Custom CSS with CSS Variables
Icons: Font Awesome 6
Fonts: Inter, Space Grotesk (Google Fonts)
Animations: CSS Keyframes & Transitions

**Design Highlights**
Color Scheme: Dark theme (#0a0a0f) with teal (#40e0d0) and blue (#4dabf7) accents
Typography: Clean and modern font pairing
Animations: Smooth transitions and floating effects
Effects: Glow effects, gradient borders, and particle backgrounds

**Customization**
Change Colors
Edit CSS variables in style.css:

css
:root {
    --primary: #40e0d0;      /* Main brand color */
    --secondary: #4dabf7;    /* Secondary color */
    --accent: #25d366;       /* Accent/CTA color */
    --dark-bg: #0a0a0f;      /* Background color */
}

**Update Content**
Company Info: Edit in index.html
Products: Update in script.js products array
Contact Details: Modify contact cards in index.html
